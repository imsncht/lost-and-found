# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-22

### Added
- User registration and login
- Lost item posting
- Found item posting
- Claim request system
- Admin claim approval/rejection
- Hibernate + MySQL integration
- JSP/Servlet MVC architecture

## [2.0.0] - 2026-04-22

### Added
- Portable persistent image upload system
- Claim proof verification fields
- Global archive/history page
- Admin dashboard with Lost / Found / Claims views
- Improved moderation workflow

### Improved
- Claim review transparency
- User trust through item lifecycle visibility
- Multi-user usability

### Fixed
- Image rendering issues in admin claims page
- Routing/path inconsistencies
