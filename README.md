# Lost & Found Management System

A Java web application for reporting lost/found items and managing claims.

## Features

- User Registration/Login
- Post Lost Items
- Post Found Items
- Claim Requests
- Admin Approval System
- MySQL + Hibernate ORM
- JSP + Servlet MVC Architecture

## Tech Stack

- Java
- Servlets
- JSP
- Hibernate
- MySQL
- Maven
- Apache Tomcat 9

## Default Admin

Email: admin@lostfound.com  
Password: password

## Run

1. Import database from sql/schema.sql
2. Configure hibernate.cfg.xml
3. Run:

mvn clean install

4. Deploy WAR on Tomcat 9
